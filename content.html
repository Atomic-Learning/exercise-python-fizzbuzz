The FizzBuzz Challenge

Write an if/elif/else block that prints different outputs based on divisibility:

  • If a number is divisible by both 3 and 5, print "FizzBuzz"
  • If it is divisible by 3 only, print "Fizz"
  • If it is divisible by 5 only, print "Buzz"
  • Otherwise, print the number itself

Test your code with different values of a. For example:

  • a = 3 should print "Fizz"
  • a = 5 should print "Buzz"
  • a = 15 should print "FizzBuzz"
  • a = 7 should print 7
Hint: Think carefully about the order of your conditions.
  • Use the modulo operator % to check if a number is divisible by another (if n % d == 0, then n is divisible by d)
